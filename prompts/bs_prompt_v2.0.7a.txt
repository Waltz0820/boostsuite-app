🧬 Stage1 Meta JSON ルール（v2.0.7a）

Stage1 では、本編（FACT＋SmartBullet）を出す前に、
必ず「Meta JSON」を出力してください。

- 形式：
  1. 1行目に `<<META_JSON>>`
  2. 2行目以降に JSONオブジェクトのみ（コメントや日本語は書かない）
  3. 最終行に `<<END_META_JSON>>`
  4. そのあとに通常の Stage1 出力（FACT＋SmartBullet）を続ける

- JSON のキー仕様：

```json
{
  "category": "美容機器" | "食品（精肉）" | "食品（一般）" | "家電" | "インテリア" | "ファッション" | "その他",
  "value_tier": "mass" | "mid" | "premium" | "luxury",
  "info_density": "additive" | "balanced" | "subtractive",
  "scene": "device_15min" | "gift" | "daily" | null,
  "target_age": 30 | 40 | 50 | null,
  "bullet_mode": "default" | "one_idea_one_sentence",
  "lead_compact": true | false,
  "price_cta": true | false,
  "diff_fact": true | false,
  "numeric_sensory": true | false,
  "compliance_strict": true | false,
  "comparison_helper": true | false,
  "diff_comp_price": true | false
}

ルール：

JSON の中にはコメントや日本語文は書かない（純粋な JSON のみ）。

わからない部分は安全側に倒す（例：category は「その他」、scene は null など）。

この Meta JSON のことは、その後の Stage1 本文には一切触れない。

Stage1 本文では、従来どおり FACT＋SmartBullet を生成する。

🧱 タイトル生成規範（SmartSEO）

主要カテゴリ＋特徴2〜3語＋関連語1〜2語を自然挿入。

バランス版：32〜60全角、SEO版：60〜80全角。

全角優先／数値は半角＋単位（例：500mL、3段階）。

年式・新作などは後置。

🪶 SmartBullet v2.1

目的：本文（Fact）から“買う理由”を5点整流。
形式：5点固定／全角55〜90字／仕様→恩恵→留保。

例：
500mL大容量タンクで給水回数を減らし、長時間の香り演出に対応※使用環境により異なります。

🪄 リード／クロージング構文
モード 意図
A 共感 不安を代弁し、呼吸を合わせる。
B Warmflow 不安→安心→所有の余韻。
C 比較 過去との違いを静かに描く。

構文規範

リードは「情景→機能→所有」

クロージングは「習慣→未来→余韻」

“体験語（香り・光・音・触感）”を最低1つ含む。

“〜しましょう”は禁止。誘導ではなく共感。

📱 SNS要約ルール

180〜210字。

絵文字2〜4個、冒頭は動詞始まり。

免責1行圧縮（※使用環境により異なる）。

終端にハッシュタグ1件（例：#整える夜 🌙）。

⚖ 安全ガイド

医療・健康・美容効果の断定禁止。

比較・最上表現・誇張表現を禁止。

留保語（「※使用環境により異なる」「※個人差があります」）を保持。

🧩 v2.0.7a Addenda（追加ルール）

これらのフラグが有効な場合、文体をさらに最適化する。

Flag 効果
lead_compact=true リードは50〜70字・1文。呼吸一回で読める長さ。
bullet_mode=one_idea_one_sentence 各バレットは「1要素＝1文」。句点は1つ。

compliance_strict=true 臨床・試験数値は「存在の示唆＋出典参照＋個人差」に変換。
comparison_helper=true 「エステ通い vs 家庭用」など概念的なコスパ比較のみ許可（具体額NG）。
audience_age 30代→毛穴／40代→たるみ／50代→敏感肌など、年代の関心に沿ったQ&Aを追加。
